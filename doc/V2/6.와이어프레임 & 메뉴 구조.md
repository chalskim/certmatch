* 카테고리 필터: 전체, 인증 컨설팅, 전문가 교육, 관련 뉴스

### 1-5. 현재 구현 상태 요약
#### ✅ 완료된 기능 (Core MVP)

- **기본 기능**: 알림, 설정, 마이페이지, 북마크

#### 🚧 개발 예정 기능 (Phase 2)
- **메시징 시스템**: 1:1 채팅, 그룹 채팅
- **결제 시스템**: 정산, 수익 관리, 결제 내역
- **고급 매칭**: AI 기반 추천, 필터링
- **관리자 기능**: 통계, 리포트, 고급 관리
- **심사 기관**: 인증 심사 관리
- **일정 관리**: 고급 캘린더, 예약 시스템

## 2. 공통 화면

### 2-1. 랜딩 페이지 (Home)
**목적:** CertMatch의 핵심 가치 소개 + 역할별 진입 유도
**구성:** ✅ *HomeScreen으로 구현됨*
* 상단 헤더: 로고 + 메뉴 + 로그인/회원가입 ✅
* 히어로 섹션:
  * 카피:
    * 메인: "인증 컨설팅 전문가를, 주·월 단위로 매칭합니다."
    * 서브: "전문가의 유휴시간을 구조화해 스타트업과 중소기업이 필요한 만큼만 전문가를 빌려 쓰는 플랫폼."
  * CTA:
    * [전문가로 시작하기]
    * [전문가 찾기]
    * [채용 공고 보기]
* 섹션: "전문가 vs 기업"의 Pain → CertMatch가 해결
* 섹션: "프리랜서 시간 패키지 / 구인 공고 두 가지 모드" 소개
* 후기/신뢰 섹션
* 푸터: 회사 정보, 약관, 개인정보처리방침, 고객센터 등

## 3. 전문가(공급자) 화면 와이어프레임

### 3-1. 전문가 대시보드
**목적:** 내 활동/수입/요청 상태 한눈에
**구성:** 🚧 *MyPagePerson으로 부분 구현됨*
* 상단 요약 카드 (4개 정도)
  * 이번 달 확정 계약 수 🚧
  * 이번 달 예상 수입 🚧
  * 진행 중 패키지 수 🚧
  * 새로운 요청 수 ✅ *알림으로 구현*
* "진행 중 패키지" 리스트
  * 패키지명 / 기업명 / 기간 / 남은 시간 / 상태 🚧
* "최근 요청" 리스트
  * 기업명 / 역할 / 예산 / 상태 🚧
* "최근 메시지" 리스트
  * 최근 대화방 3~5개 🚧

---

### 3-2. 전문가 – 내 프로필 편집
**좌측 섹션 네비:** ✅ *PersonalRegistrationEdit로 구현됨*
* 기본 정보 ✅
* 경력/프로젝트 ✅
* 포트폴리오 ✅
* 전문 분야/태그 ✅
* 단가/희망 형태 ✅

**우측 폼:** ✅ *InputExpertProfessional로 구현됨*
* 기본: 프로필 사진, 이름/닉네임, 직무, 한 줄 소개 ✅
* 경력: 회사명, 직책, 기간, 주요 성과(리스트) ✅
* 포트폴리오: 외부 링크(노션, 깃허브, 비핸스 등) ✅
* 전문 태그: 마케팅/재무/HR/PM 등 복수 선택 ✅
* 단가: 시간당 단가, 월 패키지 선호 범위 ✅
* [저장] 버튼 (상/하단) ✅

### 3-3. 전문가 – 유휴시간 설정 화면

    * 드래그로 "업무 가능 블록" 선택 🚧 *개선 예정*
  * 우:
    * 빠른 설정 버튼:
      * "평일 20–23시 전체 선택" 🚧 *개발 예정*
      * "주말 13–18시 전체 선택" 🚧 *개발 예정*
    * 현재 선택된 패턴 요약 텍스트 🚧
