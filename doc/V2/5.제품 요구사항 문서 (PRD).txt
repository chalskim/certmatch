5. CertMatch 플랫폼 – 제품 요구사항 문서 (PRD)
Product Requirements Document

1. 📄 문서 개요
항목	내용
제품명	CertMatch – 인증 매칭 플랫폼
버전	v1.0 (MVP)
작성일	2025년 10월 30일
*제품 책임자 *(Product Owner)	[이름/팀]
대상 사용자	기업(CTO, 보안담당자), 인증 컨설턴트, 심사기관, 교육기관, 심사원 지망생
목표 출시일	2026년 1분기 (3개월 내 MVP 출시)
2. 🎯 제품 비전 & 목표
2.1 제품 비전

“한국 기업의 인증 준비를 단순하고 믿을 수 있게 만드는 디지털 파트너”

2.2 핵심 목표 (OKR 기반)
Objective	Key Results (KR)
기업의 인증 여정을 획기적으로 단순화한다	- 인증 준비 평균 기간: 10개월 → 5개월 이하
- 사용자 만족도(NPS): ≥40점
- 매칭 성공률: ≥70%
인증 생태계의 신뢰 기반을 구축한다	- 검증된 컨설턴트/기관 등록률: ≥90%
- 리뷰 작성률: ≥60% 거래 완료 건
수익화 가능한 플랫폼 모델을 검증한다	- 월 활성 사용자(MAU): 1,500명
- 월 매칭 거래: 100건
- 월 수익: 2,000만 원 이상
3. 👥 타겟 사용자 페르소나
페르소나 1: *이지훈 – 스타트업 CTO *(38세)
* 목표: 투자 유치 전 ISMS-P 인증을 3개월 내 취득
* 고통:
    * “컨설팅사가 너무 많아서 누가 믿을 만한지 모르겠다.”
    * “500만 원 이상 비용은 부담된다.”
    * “정부 지원금 정보를 어디서 찾아야 할지 막막하다.”
* 기대: “한 번에 비교·견적·계약까지 가능한 원스톱 플랫폼”
페르소나 2: *박수연 – 미래의 심사원 *(32세)
* 목표: ISO 27001 심사원 자격증 취득 후 프로젝트 연결
* 고통:
    * “교육 후 실제 심사 활동 연결이 안 된다.”
    * “어떤 교육이 실무에 도움되는지 모르겠다.”
* 기대: “교육 → 자격증 → 프로젝트 매칭까지 이어지는 생태계”

4. 🧩 *핵심 기능 목록 *(MVP 기준)
기능 그룹	핵심 기능	설명	우선순위
사용자 관리	회원가입/로그인	기업, 컨설턴트, 교육기관, 개인 유형별 가입	P0
	프로필 관리	자격증, 경력, 지역, 서비스 범위 입력	P0
인증 매칭	인증 유형 선택	ISMS, ISMS-P, ISO 27001, CPPG, GS 등	P0
	예산/기간/지역 필터	사용자 요구 조건 입력	P0
	AI 기반 추천	유사 기업의 선택 패턴 기반 추천	P1
	견적 요청 & 채팅	컨설턴트와 직접 소통	P0
	계약 체결 & 결제	수수료 포함 계약서 생성 및 결제	P1
정책 연계	정부 공고 자동 수집	KISA, TTA, KFQ API 연동	P1
	지원금 자격 안내	프로필 기반 맞춤형 알림	P2
리뷰 시스템	거래 후 리뷰 작성	별점 + 텍스트 후기	P1
	평판 점수 표시	프로필에 평균 평점 노출	P1
우선순위 기준:
* P0: MVP 출시 필수
* P1: 출시 후 2개월 이내 추가
* P2: 확장 단계에서 고려


5. 📱 *사용자 플로우 *(User Flow)
5.1 기업 사용자 플로우
홈 → 인증 유형 선택 → 예산/기간/지역 입력  
→ 컨설턴트 목록 확인 → 프로필 상세 보기  
→ 견적 요청 → 채팅 → 계약 체결 → 결제 → 리뷰 작성
5.2 컨설턴트 사용자 플로우
홈 → 프로필 등록 → 서비스 조건 설정  
→ 견적 요청 수신 → 채팅 → 계약 수락  
→ 프로젝트 완료 → 리뷰 수신

6. 📐 핵심 화면 및 와이어프레임 요약
화면	주요 요소	목적

매칭 필터	예산 슬라이더, 지역 선택, 희망 기간	정확한 요구 수집
컨설턴트 목록	평점, 경력, 보유 자격증, 견적 버튼	비교 기반 의사결정
프로필 상세	후기, 실적, 보유 인증, 채팅 버튼	신뢰도 확보

마이페이지	진행 중 프로젝트, 이력, 리뷰	사용자 관리 편의성
(※ 상세 와이어프레임은 Figma/Whimsical 별도 문서로 관리)
