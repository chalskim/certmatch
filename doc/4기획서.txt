인증 매칭 플랫폼 기획서
1. 개요
목표:CPPG(개인정보보호관리사) 인증 컨설팅을 제공하는 전문가(또는 기관)와 인증을 받고자 하는 기업을 연결하는 온라인 플랫폼 구축.유형: 구인·구직 매칭 구조 기반의 B2B 서비스.

2. 주요 이해관계자
구분	설명
인증요청기업(Client)	CPPG 인증을 받고자 하는 기업. 컨설팅 의뢰를 등록함.
인증컨설턴트(Consultant)	CPPG 인증 및 보안 컨설팅 전문인. 프로젝트 지원 및 제안 가능.
관리자(Admin)	전체 회원 관리, 프로젝트 승인, 인증 과정 관리.
3. 핵심 기능
(1) 기업용 기능
* 회원가입 및 기업 정보 등록
* CPPG 인증 요청서 등록 (회사 규모, 예산, 희망 일정, 필요 분야 등)
* 매칭된 컨설턴트 확인 및 지원자 리스트 열람
* 채택 및 계약 상태 관리
* 진행 상태 모니터링 (컨설팅 단계별 진척률 표시)
* 리뷰/평가 작성
(2) 컨설턴트용 기능
* 회원가입 및 경력·자격 등록 (CPPG 인증 여부 필수 검증)
* 기업이 올린 인증 요청 프로젝트 목록 열람
* 프로젝트 지원 및 제안서 업로드
* 계약 상태 및 일정 관리
* 완료 후 리뷰 작성
(3) 관리자용 기능
* 회원 승인 및 자격 검증 (CPPG 자격증 진위 확인)
* 프로젝트 승인 / 비공개 전환
* 거래 및 정산 관리
* 통계 대시보드 (등록 수, 진행률, 만족도 등)
* 

4. 시스템 구성
모듈	설명
사용자 관리 모듈	기업/컨설턴트/관리자 권한 구분, 로그인 및 인증
프로젝트 관리 모듈	인증 요청서 작성, 지원, 계약, 리뷰
매칭 알고리즘 모듈	태그 기반(분야, 예산, 지역 등) 추천 및 매칭
결제 및 정산 모듈	컨설팅 수수료, 플랫폼 수수료 자동 정산
알림 시스템	이메일/푸시 알림, 일정 리마인더
통계 및 리포트	KPI 대시보드, 이용현황 분석
5. 기술 스택 제안
구분	기술
Frontend	React / Next.js / TailwindCSS
Backend	Node.js (NestJS) 또는 Django REST Framework
DB	PostgreSQL 또는 MySQL
Auth	JWT + OAuth2 (Google, Kakao Business 등 연동)
Infra	AWS (EC2, RDS, S3, CloudFront)
DevOps	Docker, GitHub Actions (CI/CD), Nginx
AI/추천	간단한 태그 기반 추천 → 이후 LLM 기반 자동 매칭 추천 확장 가능
6. 데이터 구조 (요약)
* User
    * id, role(기업/컨설턴트/관리자), 이름, 이메일, 인증상태
* CompanyProfile
    * user_id, 회사명, 업종, 규모, 위치, 인증이력
* ConsultantProfile
    * user_id, CPPG 자격번호, 경력, 전문분야, 포트폴리오
* Project
    * id, 기업_id, 제목, 요구사항, 예산, 마감일, 상태
* Proposal
    * id, 컨설턴트_id, 프로젝트_id, 제안서, 금액, 상태
* Review
    * 작성자_id, 대상_id, 평점, 코멘트

7. 비기능 요구사항
* 보안: HTTPS 전송, 비밀번호 해싱, 개인정보 암호화
* 확장성: 향후 ISO/ISMS 인증 등 다른 인증과 연계 가능
* 가용성: 99.9% uptime 목표
* 성능: 평균 응답시간 200ms 이하
* 정합성: 프로젝트 상태 전이(요청 → 매칭 → 계약 → 완료) 명확히 정의

8. 예상 개발 단계
단계	내용	기간(예상)
1단계	요구사항 정의 및 UI 설계	2주
2단계	백엔드 구조 설계 및 DB 구축	3주
3단계	프론트엔드 개발 및 연동	4주
4단계	테스트 및 보안 점검	2주
5단계	오픈 베타 및 피드백 반영	2주
9. 향후 확장 계획
* LLM 기반 자동 매칭 (자격·예산·리뷰 기반 추천)
* 인증 프로세스 관리 자동화 (문서 업로드, 심사 단계별 진행 현황)
* 공공기관 연동 (KISA 인증 조회 API 등)
* 해외 보안 인증 연계 (GDPR, ISO27001 등)

10. 기대 효과
* CPPG 컨설팅 시장의 비효율 해소
* 인증원과 기업 간 직접 연결로 시간·비용 절감
* 신뢰 기반의 인증 전문가 생태계 구축
