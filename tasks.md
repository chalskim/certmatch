## 📋 프로젝트 개요

**프로젝트명**: CertMatch - 인증 매칭 플랫폼
**버전**: v1.0 (MVP)
**목표 출시**: 2026년 1분기 (3개월 내)
**기술 스택**: React Native (Expo), TypeScript, Redux Toolkit, NestJS, PostgreSQL

## 🎯 핵심 목표 (OKR)

### Objective: 기업의 인증 여정을 획기적으로 단순화한다
- [ ] 인증 준비 평균 기간: 10개월 → 5개월 이하
- [ ] 사용자 만족도(NPS): ≥40점
- [ ] 매칭 성공률: ≥70%

### Objective: 인증 생태계의 신뢰 기반을 구축한다
- [ ] 검증된 컨설턴트/기관 등록률: ≥90%
- [ ] 리뷰 작성률: ≥60% 거래 완료 건

### Objective: 수익화 가능한 플랫폼 모델을 검증한다
- [ ] 월 활성 사용자(MAU): 1,500명
- [ ] 월 매칭 거래: 100건
- [ ] 월 수익: 2,000만 원 이상

#### Week 7-8: 서브 이벤트 관리
- [ ] **서브 이벤트 타입 정의 및 데이터베이스 설계**
  - [ ] SubEventType 타입 정의 (EVENT_REMINDER, DEADLINE_APPROACHING 등)
  - [ ] SubEvent 엔티티 구조 설계
  - [ ] 트리거 조건 정의 (시간 기반, 상태 기반)
  - [ ] 알림 수신자 관리

- [ ] **일정 관련 서브 이벤트**
  - [ ] 일정 리마인더 (EVENT_REMINDER) - 설정된 시간 전 푸시 알림
  - [ ] 참석자 확인 (ATTENDEE_CHECK) - 이벤트 시작 전 참석자 상태 확인
  - [ ] 일정 변경 알림 (EVENT_CHANGE) - 시간/장소 변경 시 관련자들에게 통보
  - [ ] 일정 취소 (EVENT_CANCEL) - 예약 취소 시 관련자들에게 환불 정보 전달

- [ ] **마감일정 서브 이벤트**
  - [ ] 마감 임박 알림 (DEADLINE_APPROACHING) - 마감 24시간/1시간 전 알림
  - [ ] 마감일 자동 완료 (DEADLINE_EXPIRED) - 지정된 시간 후 자동 상태 변경
  - [ ] 마감일 연장 신청 (DEADLINE_EXTENSION) - 마감일 연장 신청 가능 여부 확인

- [ ] **교육/심사 프로세스 서브 이벤트**
  - [ ] 단계별 알림 (PROCESS_STEP) - 심사/교육 각 단계 진행 상황 알림
  - [ ] 서류 제출 리마인더 (DOCUMENT_REMINDER) - 필요 서류 제출 마감 전 알림
  - [ ] 결과 통보 (RESULT_NOTIFICATION) - 합격/불합격 결과 자동 통보
  - [ ] 후속 조치 안내 (POST_ACTION) - 합격 시 다음 단계, 불합격 시 재응시 안내

#### Week 9-10: 커뮤니케이션 서브 이벤트
- [ ] **게시글 댓글 알림** (POST_COMMENT) - 내 글에 댓글 달릴 때 알림
  - [ ] 좋아요/북마크 알림 (POST_REACTION) - 게시글 반응 시 알림
  - [ ] 멘션 초대 (MENTION) - 새로운 멘션 생성 알림
  - [ ] 팔로워 신청/수락 (FOLLOW_REQUEST) - 팔로워 신청/수락 시 알림

- [ ] **채팅 관련 서브 이벤트**
  - [ ] 견적 요청 (QUOTE_REQUESTED) - 컨설턴트에게 견적 요청 시 알림
  - [ ] 견적 수락 (QUOTE_ACCEPTED) - 컨설턴트가 견적 수락 시 알림
  - [ ] 계약 요청 (CONTRACT_REQUEST) - 계약 체결 요청 시 알림
  - [ ] 계약 만료 리마인더 (CONTRACT_EXPIRY) - 계약 만료 7일/1일 전 알림

  - [ ] 일정 리마인더 (EVENT_REMINDER) - 설정된 시간 전 푸시 알림
  - [ ] 참석자 확인 (ATTENDEE_CHECK) - 이벤트 시작 전 참석자 상태 확인
  - [ ] 일정 변경 알림 (EVENT_CHANGE) - 시간/장소 변경 시 관련자들에게 통보
  - [ ] 일정 취소 (EVENT_CANCEL) - 예약 취소 시 관련자들에게 환불 정보 전달

- [ ] **마감일정 서브 이벤트**
  - [ ] 마감 임박 알림 (DEADLINE_APPROACHING) - 마감 24시간/1시간 전 알림
  - [ ] 마감일 자동 완료 (DEADLINE_EXPIRED) - 지정된 시간 후 자동 상태 변경
  - [ ] 마감일 연장 신청 (DEADLINE_EXTENSION) - 마감일 연장 신청 가능 여부 확인

- [ ] **교육/심사 프로세스 서브 이벤트**
  - [ ] 단계별 알림 (PROCESS_STEP) - 심사/교육 각 단계 진행 상황 알림
  - [ ] 서류 제출 리마인더 (DOCUMENT_REMINDER) - 필요 서류 제출 마감 전 알림
  - [ ] 결과 통보 (RESULT_NOTIFICATION) - 합격/불합격 결과 자동 통보
  - [ ] 후속 조치 안내 (POST_ACTION) - 합격 시 다음 단계, 불합격 시 재응시 안내

#### Week 9-10: 커뮤니케이션 서브 이벤트
